**For**: kids (4+)

# Lego Beats 🧱

A 2D block sequencer. Tap the colored blocks to activate them — the white cursor sweeps left to right and plays every lit block it passes, looping endlessly.

## How to play

- **Tap** any block to toggle it on (lights up) or off
- **Drag** across empty blocks to paint a run of notes
- The white cursor sweeps left → right, playing each active block as it passes
- **−** / **+** buttons adjust tempo (40–160 BPM)
- **✕ Clear** erases all blocks so you can start fresh

## Design notes

- Inspired by BrickMusicTable (arxiv 2411.13224, Nov 2024): tangible lego brick grid sequencer tested with 150+ children aged 3–13. Key finding: construction-as-composition is naturally intuitive — place = note on, remove = note off.
- Visual: lego-style bricks with top-sheen + center stud. Bounce animation on play.
- Zero permissions needed. No reading required.
- Pentatonic tuning ensures every combination sounds musical.
